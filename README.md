# DeDe
DeDe is a lifelong cognitively revisable AI companion. Built around a multi-agent cognitive architecture (Gnosis, Nous, Doxa, Reduction, NOUSCOPE, Memory), it aims to promote understanding rather than certainty, helping users explore, question, learn, and continuously refine their mental models.

# DeDe

**Your Deliberative Daimon**  
*A lifelong cognitively revisable AI companion.*

DeDe is an experimental AI project built around a multi-agent cognitive architecture.  
Its purpose is not only to answer, but to help users think, question, revise, and refine their understanding over time.

## Core Architecture

- **Gnosis Agent** — knowledge, evidence, sources
- **Nous Agent** — integrated understanding
- **Doxa Agent** — certainty and cognitive closure
- **Reduction Agent** — hidden assumptions and blind spots
- **NOUSCOPE Agent** — user cognitive filter
- **Memory Agent** — long-term continuity
- **Daimon Orchestrator** — coordination of all agents

## Status

DeDe is currently in early development.

DOXA Detector, NOUSCOPE, and related cognitive modules are under active research and calibration.

## Vision

The goal is to build an AI companion that remains cognitively revisable instead of simply reinforcing existing beliefs.

The LLM is the engine of language.  
DeDe is the architecture of cognitive calibration.
