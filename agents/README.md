# Cognitive Agents

The `agents` module contains the specialized cognitive agents that collectively form DeDe's multi-agent reasoning architecture.

Each agent is responsible for a distinct dimension of cognition rather than attempting to solve every problem alone.

Instead of relying on a single monolithic reasoning process, DeDe distributes cognitive tasks across complementary agents:

- **Gnosis** — Knowledge acquisition and verification.
- **Nous** — Integrated understanding and synthesis.
- **Doxa** — Certainty, cognitive closure, and overconfidence analysis.
- **Reduction** — Detection of assumptions and hidden reductions.
- **NOUSCOPE** — Modeling of cognitive filters.
- **Cognitive Therapy** — Cognitive recalibration and revisability support.

Their analyses are coordinated by the **Daimon Orchestrator**, which integrates the contributions of every cognitive agent into a coherent response.

This modular architecture allows new cognitive agents to be added as the DeDe ecosystem evolves while preserving transparency, extensibility, and long-term cognitive continuity.
