# Core

The `core` module contains the central infrastructure of DeDe.

Unlike the specialized cognitive agents, the core components are responsible for coordinating the entire cognitive architecture rather than performing domain-specific reasoning.

The most important component is the **Daimon Orchestrator**, which acts as the cognitive brain of DeDe. It receives user input, retrieves relevant long-term memory, coordinates the specialized cognitive agents, integrates their analyses, and produces a coherent final response.

As the project evolves, the core module will also include components responsible for:

- Conversation management
- Agent orchestration
- Inter-agent communication
- Context management
- Task scheduling
- Cognitive workflow execution
- Long-term cognitive continuity

The core layer provides the foundation upon which every cognitive capability of DeDe is built. It ensures that specialized agents operate as a unified cognitive system rather than as independent modules.
